# Changelog

このプロジェクトの変更履歴を記録します。

## [1.1.0] - 2026-07-06

### 追加

- コピペ読み込みのトグル内（テキストボックス上）に「プレイデータCSVのダウンロードはこちらから」リンクを追加
  - 最近プレーしていない曲: 「コピペで読み込む」内
  - バージョン間スコア比較: 「現行バージョンをコピペで読み込む」内

### 変更

- バージョン間スコア比較の「比較する」ボタンを削除
  - CSV読み込み時・レベル選択時・チェックボックス変更時に自動で比較を実行するように変更
  - 「過去データをクリア」時は比較結果もクリアするように変更

## [1.0.0] - 2026-07-06

### 追加

- **アプリ初版作成**（単一HTML・サーバー不要・データはブラウザ内処理のみ）
  - 「最近プレーしていない曲」タブ: プレーデータCSVを読み込み、最終プレーから指定か月数以上経過した曲を一覧表示
  - 「バージョン間スコア比較」タブ: 現行/過去バージョンのCSVを比較し、過去の自己ベストよりスコアが下がっている譜面を抽出
  - ランダム4曲選曲機能（今日の課題曲）
- CSV読み込みを2方式に対応（ファイル選択 / コピペ貼り付け）※スマホ対応
- 文字コード自動判別（UTF-8 / Shift_JIS）
- レベル（☆1〜12）絞り込み機能を両タブに追加
- クリアランプ表示（曲名左のカラーバー、8色）
  - FULLCOMBO `#f9c` / EXH `#FFE375` / HARD `#DD5454` / CLEAR `#5EEAFF` / EASY `#BDFF7C` / ASSIST `#9474DD` / FAILED `#A9A9A9` / NO PLAY `#ffffff`
  - バージョン間スコア比較では現行バージョンCSVのランプを参照
- 「今作未プレーの譜面も表示」「未プレーの譜面も表示」チェックボックスを追加
  - 今作未プレー: ランプあり かつ スコア0（過去作でのみプレー）
  - 未プレー: ランプが NO PLAY
  - 該当曲は経過日数欄に「今作未プレー」「未プレー」と表示
- 使い方説明（CSV取得方法・公式ダウンロードページへのリンク）を追加
- 非公式ファンツールである旨・商標・プライバシーの注記フッターを追加
- PWA対応（manifest / Service Worker / ホーム画面用アイコン）
- 公開用パッケージ `iidx-app/` を作成（index.html 化）

### 変更

- 絞り込み条件のUIを縦並びに統一（ラベル位置揃え）
- 絞り込みボタンを廃止し、全条件を即時反映に変更
- 収録バージョンでの絞り込み機能を削除
- 「プレー回数0の曲を除外」チェックボックスを削除（スコア判定に統合したため）
- レベル未選択時は結果を表示しない仕様に変更（両タブ。選択後は即時反映）
- 検索結果の列構成を変更
  - 最近プレーしていない曲: 経過日数 / タイトル / 回数 / 最終プレー日時
  - バージョン間スコア比較: ☆列を削除、難易度表示を頭文字1文字（B/N/H/A/L）に変更
- 検索結果セルを中央揃え、曲名のみ左揃えに変更
- ランダム選曲の対象を「スコアが付いている譜面を持つ曲」に限定
- 検索結果から「スコア0の曲」を除外（他難易度プレーでプレー回数のみ加算されるCSV仕様への対応）

### 修正

- スマホ表示の改善
  - 表の横スクロールを廃止し画面幅に収まるよう修正（長い曲名は折り返し）
  - 「最終プレー日時」見出しのはみ出しを修正
  - タブを横幅50%ずつに調整し、各要素の左端を統一
- スクロール時に曲名が見出しに重なる問題を修正（z-index）
- 「未プレー」表示ON/OFFで比較結果の列幅がずれる問題を修正（列幅固定）
